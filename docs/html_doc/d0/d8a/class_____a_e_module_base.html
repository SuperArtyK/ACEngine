Detailed Description

template<typename T>
class __AEModuleBase< T >

This is a base class for all basic modules, meaning that the module is not using other modules to work.

Note
Intended to be inherited from in class declaration, not direct usage. Usage: class [classname] : public __AEModuleBase<[classname]>
Warning
If you do not include REGISTER_MODULE(T) after the whole class definition (before closing '}'), the program won't compile
If you do not include COPYCONSTRUCT_MODULE(T, val) in the copy-constructor, the module increment (every constructing action) won't work!

TInheriting module's class name
