Detailed Description

This file contains the results for the trigonometric function lookup tables. These will be insanely fast, almost as the direct memory access (since just value in array), but won't offer much of precision – within 1 degree. (Un)Define ENGINE_ENABLE_MATH_LOOKUP flag in AEFlags.hpp to enable/disable this.
