120 /// <summary>
121 /// Returns the **index to the trig table** from the given (int) degree number.
122 /// @remark If the number is negative, it wraps it from the end (-1 is the last item of the table)
123 /// @note Not recommended to use directly. It's a helper function for the trig lookup table functions

126 /// <returns>
