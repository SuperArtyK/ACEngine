Detailed Description

ArtyK's Engine File Writer; umm, it writes data to given file.

Just create it and dump the gigabytes of data to your files. Hungarian notation is fw. Flags start with AEFW_

Warning
This is not thread safe!

Open(and create) file with given name and flag.

On success sets AEFileWriter::m_cFlags to passed flag value.

Note
If the provided file flag is invalid, it returns false and sets the AEFW_ERR_FILE_WRONG_FLAG as the last error
Warning
Using the AEFW_FLAG_APPEND_NO_CURSOR_MOVE disables AEFileWriter::getCursorPos(), AEFileWriter::setCursorPos(), and AEFileWriter::getFileSize(), since they manipulate the cursor.
Parameters
strName of the file, with extension
flagsFlags for file opening, AEFW_FLAG_* macros. More info in the docs
af_intervalInterval in file writes between automatic file flushing
Returns
AEFW_ERR_NOERROR if the file was opened without errors, other error flag value otherwise
