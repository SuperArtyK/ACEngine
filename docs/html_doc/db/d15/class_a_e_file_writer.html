Detailed Description

ArtyK's Engine File Writer.

Umm, it writes data to given file. Just create it and dump the gigabytes of data to your files. Hungarian notation is fw Flags start with AEFW_

Sets read cursor position to pos from origin.

Note
If cursor is beyond eof, it fills space between eof and cursor with null-bytes when data is written.
Warning
Fails and returns AEFW_ERR_FILE_WRONG_FLAG if the flag used to open the current file is AEFW_FLAG_APPEND_NO_CURSOR_MOVE
Parameters
posPosition to be set to relative to origin (same as "offset" in fseek)
originRelative origin for the operation. Google SEEK_SET, SEEK_CUR and SEEK_END for more details
Returns
0 on success, AEFW_ERR_NOT_OPEN if file's closed, or other things that fseek can return
