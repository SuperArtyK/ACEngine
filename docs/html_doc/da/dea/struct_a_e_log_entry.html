inlinestatic

Allocates the queue of the given size on the heap and returns the pointer to it's first node.

Optionally may loop the newly-allocated queue to the old queue.

Note
You should delete[] the pointer after you're done using it (unless you like mem-leaks)
If the amt is 0, throws the std::runtime exception
Parameters
amtThe amount of entries in the queue(size)
oldqueueThe pointer to the old queue to loop the new queue to.
loopQueueThe flag to indicate whether to loop the generated queue at all
Returns
Pointer to the first node of the allocated queue
