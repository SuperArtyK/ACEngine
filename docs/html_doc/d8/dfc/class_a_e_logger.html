Detailed Description

The ArtyK's Engine's Logger module – it manages the writing to the log files.

The log is written by requesting and filling the entry in the queue. The AELogger instance launches the separate thread that looks at the entries in the queue, retrieves and formats the data in them, and writes it to the file. Afterwards that entry in the queue is cleared. The queue can expand if it's too little. But....I don't know how to shrink it.
